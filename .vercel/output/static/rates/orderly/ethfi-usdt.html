How Funding Works on Orderly Network

Orderly Network calculates funding rates hourly using a shared order book mechanism that aggregates positions across all connected frontends. The platform's infrastructure layer ensures synchronized funding payments regardless of which frontend interface traders use to access ETHFI/USDT perpetuals. Funding rates are determined by the premium/discount between the perpetual price and ETHFI spot price across the network's liquidity sources.
